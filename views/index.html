Final Project

This project focused primarily on building a backend API with Node.js; thus, a front end was not fully built. However, the following calls work in the address bar:

Get Accepted Requests (:states - must be an abbreviation of a state)

  • /states/
  • /states/?contig=true
  • /states/?contig=false
  • /states/:state/funfact
  • /states/:state/capital
  • /states/:state/nickname
  • /states/:state/population
  • /states/:state/admission
